Help

This randomizer targets the original NTSC-U release of Castlevania: Symphony of the Night (SLU-00067). You will need an image of your game disc in BIN/CUE format. If your disc image is split across two .bin files, drag only the first .bin file (track 1) into the drop zone.

If the seed name is appended to your randomized image filename after downloading, you will need to modify your cue file track listing to reference the new filename.
